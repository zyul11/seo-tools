💡 Alt Text Best Practices

  • Describe the image content accurately — don't just stuff keywords
  • Keep alt text under 125 characters (screen readers cut off after that)
  • Decorative images should use alt="" (empty) — not missing
  • Include relevant keywords naturally when they fit the image description
  • Every <img> must have an alt attribute for WCAG compliance
