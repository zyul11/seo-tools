💡 Page Size Guidelines

  • < 100 KB → Excellent. Lightweight page loads fast on all connections.
  • 100-300 KB → Good. Standard page weight for most sites.
  • 300-500 KB → Fair. Consider trimming unused CSS/JS.
  • 500 KB - 1 MB → Heavy. Compress images, minify assets.
  • > 1 MB → Too heavy. Major optimization needed for good Core Web Vitals.
  • Keep code-to-content ratio under 50% for better SEO performance.
